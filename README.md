# HouseFix
Site desenvolvido seguindo um [design pronto](https://www.figma.com/community/file/1479561357645494407) de um site, feito por [Al Razi Siam](https://www.figma.com/@alrazisiam) no Figma.

O objetivo deste projeto é aprender a usar Bootstrap. A idéia foi fazer um site simples, sem usar nenhum framework, e usar o CSS e JS do Bootstrap através dos links CDN.
